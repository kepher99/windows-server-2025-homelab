# Windows Server 2025 Homelab on VMware Workstation Pro / VirtualBox

## Overview
This repository documents the deployment of a Windows Server 2025 homelab environment using VMware Workstation Pro and VirtualBox. The purpose of this lab is to build practical skills in Windows Server administration, virtualization, IT support, and enterprise infrastructure configuration.

## Objectives
- Install and configure Windows Server 2025 in a virtual environment
- Practice virtualization using VMware Workstation Pro and VirtualBox
- Build a foundation for Active Directory, DNS, DHCP, and Group Policy labs
- Simulate real-world IT administration and troubleshooting tasks
- Maintain professional technical documentation for repeatability and portfolio use

## Lab Environment
- **Host Platform:** Personal workstation/laptop
- **Virtualization Tools:** VMware Workstation Pro, VirtualBox
- **Guest OS:** Windows Server 2025
- **Planned Client Systems:** Windows 10, Windows 11
